# 多语言参考图生成说明

本说明对应 `LanguageSortValidation` 组件下方「参考图」资源：`assets/resources/references/case-01.png` … `case-NN.png`。

## 用途

- 在 [texteditor.cn](https://texteditor.cn/) 富文本编辑区中输入与用例相同的文本，截取**仅可编辑区域**（不含顶部工具栏、底部按钮）。
- 本地再按文字内容做**二次裁剪**（去掉蓝色边框等 UI 噪声），得到紧凑参考图。
- 与 Cocos 内 Label 实际渲染对照，便于测试人员肉眼校验混排、换行、RTL、Emoji 等。

## 环境要求

- Node.js（建议 18+）
- 项目根目录已安装依赖：

```bash
cd /path/to/LanguaeTest
npm install
npx playwright install chromium
```

若终端配置了代理且 `curl`/`npm` 失败，可先取消代理再执行：

```bash
env -u http_proxy -u https_proxy -u HTTP_PROXY -u HTTPS_PROXY npm install
```

## 用例文本从哪里来改

编辑 **`scripts/text_cases.json`**：

- 支持格式一：对象数组（推荐，可带 `title` 备注）

```json
[
  { "title": "中文-简体", "text": "你的文本…" },
  { "title": "英文", "text": "Another line\n第二行" }
]
```

- 支持格式二：纯字符串数组

```json
["第一条例文", "第二条例文"]
```

**顺序**决定输出文件名：`第 1 条` → `case-01.png`，以此类推。  
请与 `assets/LanguageSortValidation.ts` 里 `_cases` 的顺序保持一致，否则对照会错位。

多行文本在 JSON 里使用 `\n` 表示换行。

## 一键生成命令（项目根目录）

| 命令 | 作用 |
|------|------|
| `npm run capture-refs` | 截图 + 裁剪（推荐日常使用） |
| `npm run capture-refs:raw` | 仅 Playwright 截图，不裁剪 |
| `npm run capture-refs:crop` | 仅对已有 `case-*.png` 做裁剪 |

输出目录固定为：

`assets/resources/references/case-01.png` … `case-NN.png`

## 裁剪逻辑说明（`scripts/crop_reference_images.js`）

- 将接近纯白背景（RGB ≥ 245）视为背景。
- 扫描时**忽略四周边缘约 10px**，避免把编辑区蓝色边框当成「前景」导致裁不出紧凑区域。
- 在文字包围盒外再扩 **`PADDING`（默认 24px）** 留白。

若某站改版导致边框更粗，可适当增大脚本里的 `SCAN_MARGIN` 或 `PADDING`。

## 在 Cocos Creator 里生效

1. 生成或替换 PNG 后，在资源管理器中选中 **`assets/resources/references`**，执行**刷新 / 重新导入**。
2. 确保图片在编辑器中被识别为 **`SpriteFrame`**（或依赖脚本里对 `Texture2D` 的兼容包装）；若仍为纯 Texture，请在资源检查器中改为 **Sprite(2D)** 类型。
3. `LanguageSortValidation` 中：
   - `referenceResourceDir` 默认为 `references`（对应 `assets/resources/references`）
   - `autoLoadReferenceFrames` 开启后会自动加载该目录下资源

## 其他脚本

- **`scripts/import_reference_images.sh`**：将某目录下图片批量拷贝并重命名为 `case-01.png` 起（适合手工准备的图）。
- **`scripts/download_reference_images.sh`**：示例用 `curl` 下载外链图（受本机代理/网络影响较大，可改用本流程）。

## 常见问题

**控制台 `loaded=0`、参考图不显示**

- 新 PNG 未生成 `.meta`：在 Cocos 中刷新 `references` 目录。
- 资源类型不是 SpriteFrame：按上文改为 Sprite 或依赖脚本内 Texture 包装逻辑。

**Playwright 打不开网页或超时**

- 检查网络、防火墙；必要时在无代理环境下执行命令。

**参考图与 Label 仍不一致**

- 浏览器使用系统/网页字体，Cocos Label 使用项目字体，**像素级不可能完全一致**；本流程侧重「排版与字符是否明显异常」的对照。
